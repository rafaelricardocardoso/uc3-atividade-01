markdown oque é e como usar

é uma linguagem simples de marcação

Markdown converte seu texto em HTML válido

Markdown é frequentemente usado para formatar arquivos README, para escrever mensagens em fóruns de discussão online e para criar rich text

como usar:

  • para fazer a titulação você usará
  • # Título
  • ## Título h2
  • ### Título h3
  • #### Título h4
  • ##### Título h5
  • ###### Título h6
  • negrito **
  • itálico *
  • tachado ~~

oque é css e como usar

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style . Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos. h2 h1 body html h2

como usar:

crie uma pasta chamada assets e dentro dela crie outra pasta chamada css e crie um arquivo.css e coloque o seu style dentro desse arquivo

oque é html e como usar

HTML abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web.

para usar o html você precisará de tempo para aprender todos os códigos porém o basico é (html) (head) (body) (p) (h1) etc...
